# 444 Custom Print Hub 🖨️

> “At 444 Custom Print Hub, we are committed to delivering exceptional quality and service to meet all your printing and design needs.”

A clean, modern, and UX-friendly **marketing (pseudo-ecommerce)** website built for the **444 Custom Print Hub** — a local printing business based in **Antipolo, Rizal**.

---

## 🧰 Tech Stack

- **Next.js 15** (App Router, Server Components, RSC)
- **React 19** (with Server Actions)
- **Tailwind CSS v4**
- **Framer Motion** for smooth UI transitions
- **Shadcn UI** components for accessible and customizable UI primitives

---

## ✨ Features

- 🖥️ **Landing Page** – elegant hero layout, strong CTA, and overview of the brand
- 🔁 **Infinite Scrolling** – continuous product loading with pagination fallback
- 📦 **Card Components & Skeletons** – consistent category and product visuals with loading placeholders
- ⏳ **Loading Handlers** – optimized UX while waiting for data or page transitions
- 🧃 **Filter Products** – dynamic filtering by category  
  (currently: **Ref Magnet** and **Candle**)
- 🔍 **Product Page** – individual product details with clean layout, gallery tab, and responsive design

---

## 📍 Location

Developed for a real business operating in:
